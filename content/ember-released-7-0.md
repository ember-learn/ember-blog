The Ember project is excited to announce the release of Ember v7.0. Following [Ember's Major Version Policy](https://rfcs.emberjs.com/id/0830-evolving-embers-major-version-process), version 7.0 will be the first release of the 7.x series. This release of Ember.js means the previous version, 6.12, is now an LTS (Long Term Support) version.
